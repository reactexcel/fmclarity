Class: Action

core/Actions.Action

An Action is a state-changing interaction available to the app user which is wrapped with authentication. It is a primary structure used for role based access control.

Constructor

new Action(name, action, typeopt, pathopt, labelopt, iconopt, descriptionopt)

Create an Action.
Parameters:
Name Type Attributes Description
name string The registered name of this action.
action function A function with the code for the action to be performed.
type string <optional>
 The type of model this action expects ie 'request', 'facility'
path string <optional>
 The url of this action, if provided then running the action changes the url to this.
label string <optional>
 Human readable name of the action used by interface elements.
icon string <optional>
 A css class that can be used to display an icon related to this action (used by interface elements).
description string <optional>
 A description of this action to be used by interface elements
