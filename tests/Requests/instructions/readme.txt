Pre-requisites before runnin Selenium Tests:

1. Install Selenium IDE into Firefox
	from https://addons.mozilla.org/en-US/firefox/addon/selenium-ide/
2. Install Selenium IDE Flow-control
	from https://addons.mozilla.org/en-US/firefox/addon/flow-control/
